Необходимо разработать веб-приложение для продажи велосипедов со следующим функционалом:
1. Каталог товаров
Отображение велосипедов в виде карточек с фото, названием, ценой и кратким описанием

Возможность фильтрации товаров: 
По типу велосипеда (горный, шоссейный, городской и т.д.)
По производителю
По ценовому диапазону
По размеру рамы
По наличию на складе

Возможность сортировки:
По цене (дешевле/дороже)
По популярности
По рейтингу
По новизне

Кнопка быстрого добавления в корзину на каждой карточке

2. Корзина
- Список выбранных товаров
- Возможность изменить количество каждого товара
- Удаление товаров из корзины
- Отображение промежуточной суммы
- Поле для ввода промокода
- Отображение итоговой суммы с учетом скидок

3. Оформление заказа
Форма с валидацией для ввода данных покупателя:
ФИО
Email
Телефон
Адрес доставки

Выбор способа доставки:
Самовывоз
Курьерская доставка
Почта России

Выбор способа оплаты:
Онлайн
Наличными при получении
Картой при получении

4. Переключатель между темной и светлой темой


данные Велосипедов придумать или использовать моки, которые скину или подгружать 
файлик json

Среди технологий использовать: react, formik(для валидации), redux toolkit, 
styled components(для стилизации), typescript
